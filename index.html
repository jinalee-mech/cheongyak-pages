청약은 "당첨만 되면 끝"인 복권이 아니라, 내 점수와 자격으로 어디가 현실적인지를 찾는 문제입니다. 아래에 조건을 입력하면 진행 중인 공고를 가점제 · 추첨제 · 특별공급 세 갈래의 전략으로 정리합니다.

아래 모든 단지명·수치는 데모용 가상 샘플입니다 — FastAPI 백엔드 구현 시 청약홈 Open API 데이터를 같은 화면 구조로 연결하도록 설계

01

시장 현황 — 양극화를 숫자로

"당첨 = 수익"이라는 전제는 일부 단지에만 성립합니다. 평균 아래에는 미달 단지가, 평균 위에는 점수로 닿을 수 없는 단지가 함께 있습니다. 그래서 분석이 필요합니다.

02

내 조건 입력

무주택기간과 부양가족 수는 산정 기준이 까다로워 자동으로 추정하지 않습니다 — 아래 도움말을 참고해 직접 입력해 주세요.

무주택기간 산정 기준과 신혼부부 특공 검토에 사용

예: 5년 = 60. 6개월 미만이면 그대로 입력 (1점 처리)

만 30세 도달일(그 전에 혼인했다면 혼인신고일)부터 산정. 1년 미만이면 0 입력

만 30세 미만 미혼은 산정 기간이 시작되지 않아 무주택기간 가점이 0점으로 계산됩니다

동일 주민등록표 기준 — 직계존속은 3년 이상 계속 등재 등 조건 확인 필요

유주택이면 무주택 가점 0점, 일부 전형은 신청 제한
